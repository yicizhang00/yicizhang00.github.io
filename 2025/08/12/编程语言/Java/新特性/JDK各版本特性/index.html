  • ејҖеҸ‘иҖ…еҸҜд»Ҙж №жҚ®еә”з”ЁйңҖжұӮеҸӘеј•е…ҘйңҖиҰҒзҡ„жЁЎеқ—пјҢеҮҸе°ҸдҪ“з§Ҝ

еҗҜеҠЁжЁЎеқ—еҢ–еә”з”Ё

G1 жҲҗдёәй»ҳи®Өеһғеңҫеӣһж”¶еҷЁпјҲGCпјү

JDK9 ејҖе§ӢпјҢG1пјҲGarbage-Firstпјүеһғеңҫеӣһж”¶еҷЁжӯЈејҸеҸ–д»Ј Parallel GCпјҢжҲҗдёә Java иҷҡжӢҹжңәзҡ„й»ҳи®Өеһғеңҫеӣһж”¶еҷЁгҖӮG1 дјҳе…Ҳж»Ўи¶ідҪҺе»¶иҝҹгҖҒй«ҳеҗһеҗҗзҡ„жңҚеҠЎз«ҜеңәжҷҜпјҢйҖӮеҗҲеӨ§еҶ…еӯҳгҖҒеӨҡж ёеӨ„зҗҶеҷЁзҡ„зҺ°д»Јеә”з”ЁгҖӮ

дё»иҰҒзү№жҖ§

  • ж”ҜжҢҒеӨҡж ёе№¶еҸ‘гҖҒдҪҺжҡӮеҒң GC
  • еҸҜд»ҘйҖҡиҝҮ -XX:MaxGCPauseMillis жҺ§еҲ¶жңҖеӨ§жҡӮеҒңж—¶й—ҙ
  • жҢүвҖңеҲҶеҢәвҖқеҶ…еӯҳеҢәеҹҹпјҢзҒөжҙ»й«ҳж•Ҳең°е№¶иЎҢ/е№¶еҸ‘ж”¶йӣҶ
  • дёҚеҶҚйңҖиҰҒжүӢеҠЁжҢҮе®ҡ -XX:+UseG1GCпјҢз§»йҷӨеҗҺй»ҳи®ӨйҮҮз”Ё G1

д»Јз ҒдҪ“йӘҢ

дёҖиҲ¬дёҚйңҖиҰҒжӣҙж”№д»Јз ҒпјҢд»… VM еҸӮж•°дјҡеҲҮжҚўгҖӮеҰӮйңҖи§ӮеҜҹпјҢеҗҜеҠЁеҸӮж•°еҸҜеҠ пјҡ

жүӢеҠЁжҢҮе®ҡдҪҝз”Ёе…¶е®ғ GC

JDK21 жӯЈејҸеј•е…ҘиҷҡжӢҹзәҝзЁӢпјҲvirtual threadsпјүпјҢиҝҷжҳҜ Java е№¶еҸ‘жЁЎеһӢзҡ„дёҖж¬ЎйҮҚиҰҒйқ©ж–°гҖӮиҷҡжӢҹзәҝзЁӢжҳҜдёҖз§ҚиҪ»йҮҸзә§зәҝзЁӢпјҢз”ұ Java иҷҡжӢҹжңәпјҲJVMпјүи°ғеәҰпјҢеҸҜд»ҘеӨ§е№…жҸҗеҚҮй«ҳе№¶еҸ‘еңәжҷҜдёӢзҡ„еҗһеҗҗдёҺиө„жәҗеҲ©з”ЁзҺҮпјҢжһҒеӨ§з®ҖеҢ–й«ҳе№¶еҸ‘жңҚеҠЎеҷЁзј–еҶҷйҡҫеәҰгҖӮ

дё»иҰҒзү№жҖ§

  • иҷҡжӢҹзәҝзЁӢжҳҜ Thread зҡ„дёҖз§Қе®һзҺ°пјҲjava.lang.ThreadпјүпјҢдёҺе№іеҸ°зәҝзЁӢжӢҘжңүеҹәжң¬дёҖиҮҙзҡ„ APIгҖӮ
  • еҲӣе»әе’Ңй”ҖжҜҒиҷҡжӢҹзәҝзЁӢзҡ„д»Јд»·жһҒдҪҺпјҢйҖҡеёёиғҪиҪ»жқҫеҲӣе»әж•°дёҮз”ҡиҮізҷҫдёҮзә§зҡ„иҷҡжӢҹзәҝзЁӢгҖӮ
  • и®©йҳ»еЎһеһӢд»Јз ҒпјҲеҰӮ I/OгҖҒsleep зӯүпјүдёҺејӮжӯҘзј–зЁӢдёҖж ·й«ҳж•ҲпјҢдҪҶд»Јз ҒйЈҺж јд»ҚдёәеҗҢжӯҘж–№ејҸпјҢжҳ“иҜ»жҳ“з»ҙжҠӨгҖӮ
  • зәҝзЁӢи°ғеәҰдәӨз”ұ JVMпјҢиғҢеҗҺйҖҡеёёдёҺе№іеҸ°зәҝзЁӢжұ з»‘е®ҡпјҢдҪҶз”ЁжҲ·ж— йңҖе…іеҝғгҖӮ

  • иғҪз”ЁвҖңзңӢиө·жқҘйҳ»еЎһгҖҒе®һеҲҷи°ғеәҰй«ҳж•ҲвҖқзҡ„ж–№ејҸзј–еҶҷй«ҳе№¶еҸ‘зҪ‘з»ңеә”з”ЁзЁӢеәҸпјҲеҰӮ Web жңҚеҠЎеҷЁпјүгҖӮ
  • еӨ§е№…еҮҸе°‘еӨҚжқӮвҖңеӣһи°ғең°зӢұвҖқжҲ–ејӮжӯҘд»Јз ҒйңҖжұӮгҖӮ
  • жҸҗй«ҳеә”з”Ёзҡ„еҸҜз»ҙжҠӨжҖ§гҖҒжү©еұ•жҖ§е’ҢеҗһеҗҗиғҪеҠӣгҖӮ

иҷҡжӢҹзәҝзЁӢзҡ„жҸҗеҮәж Үеҝ—зқҖ Java еңЁе№¶еҸ‘зј–зЁӢйўҶеҹҹиҝҲеҗ‘ Go д№Ӣзұ»вҖңзҷҫдёҮеҚҸзЁӢвҖқж—¶д»Јзҡ„иғҪеҠӣпјҢеҗҢж—¶дҝқз•ҷдәҶдёҘи°Ёзҡ„зұ»еһӢзі»з»ҹе’ҢзҺ°жңүз”ҹжҖҒзҡ„е…је®№жҖ§гҖӮ

еҲҶд»ЈZGC

JDK21 еј•е…Ҙзҡ„еҲҶд»ЈZGCпјҲGenerational ZGCпјү

еҲҶд»ЈZGCпјҲGenerational ZGCпјүжҳҜеңЁеҺҹжңү ZGC еһғеңҫеӣһж”¶еҷЁеҹәзЎҖдёҠе®һзҺ°зҡ„еҲҶд»ЈпјҲGenerationalпјүеҠҹиғҪпјҢиҝӣдёҖжӯҘжҸҗеҚҮдәҶдҪҺе»¶иҝҹеһғеңҫеӣһж”¶зҡ„ж•ҲзҺҮпјҢзү№еҲ«жҳҜеңЁеӨ§еҜ№иұЎгҖҒж–°з”ҹд»ЈеҜ№иұЎз”ҹе‘Ҫе‘Ёжңҹйў‘з№ҒеҸҳеҠЁзҡ„еә”з”ЁеңәжҷҜгҖӮ

дё»иҰҒзү№жҖ§

  • ж”ҜжҢҒе°Ҷе ҶеҲ’еҲҶдёәж–°з”ҹд»ЈпјҲYoung Generationпјүе’ҢиҖҒе№ҙд»ЈпјҲOld Generationпјү
  • еҜ№еӨ§еӨҡж•°зҹӯе‘ҪеҜ№иұЎиҝӣиЎҢжӣҙй«ҳж•Ҳзҡ„еҶ…еӯҳеӣһж”¶пјҢеҮҸе°‘ Full GC еҸ‘з”ҹйў‘зҺҮ
  • дҝқжҢҒ ZGC жң¬иә«зҡ„дҪҺжҡӮеҒңпјҲдәҡжҜ«з§’зә§пјүе’ҢеҸҜжү©еұ•жҖ§
  • жҸҗеҚҮдәҶеҗһеҗҗйҮҸпјҢйҷҚдҪҺдәҶеә”з”Ёе»¶иҝҹ
